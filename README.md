🎮 Text Base — "A Text RPG"
Because life is random, but it's what you make of it.
---
What is Text Base?
Text Base is an original text-based RPG built from the ground up by Blake Evan Davis starting November 2018.
It's not your average RPG. Forget polished graphics and predefined story paths handed to you by a billion dollar studio. This is raw, comedic, and unapologetically random — just like life.
The game is built on a comedic friendship — full of absurd humor, ridiculous weapons, strange characters, and moments that will make you laugh out loud. Nothing in this game is meant to be taken seriously. It's all in good fun.
> *"I created this game to remember that life is random — but it's what you make of it."*
> — Blake Evan Davis, Creator
---
🎯 Features
⚔️ Turn-based combat — fight monsters ranging from Gross Goblins to Three Headed Dragons
🗺️ Fast Travel — explore locations like Cancer City, Sticky Swamp, and Ancient Castle
🛒 Trading system — buy weapons, armor, and potions from wandering merchants
🏟️ Arena mode — test yourself from Easy to Legendary difficulty
🎒 Inventory & crafting — collect materials and craft your own gear
📈 Level & XP system — grow stronger as you play
😂 Unhinged humor — written with real friends, for real laughs
🤖 AI Dialog System (coming soon) — shape your character using your own words and decisions

⚠️ Content Warning
This game contains vulgar jokes and adult humor. If you can't take a joke, this might not be for you. Everything said in this game is fictional and intended purely for comedic entertainment.

🚀 How to Play
Requirements: Python 3.x
```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/textbase.git

# Navigate to the folder
cd textbase

# Run the game
python textBase.py
```
That's it. No installs, no dependencies, no nonsense. Just Python and your keyboard.
