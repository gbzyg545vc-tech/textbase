What is Text Base?
Text Base is an original web-based text RPG built from the ground up by Blake Evan Davis starting November 2018.
Forget polished graphics and predefined story paths handed to you by a billion dollar studio. This is raw, comedic, and unapologetically random — just like life.
Built on a real friendship — full of absurd humor, ridiculous weapons, strange characters, and moments that will make you laugh out loud. Nothing in this game is meant to be taken seriously. It's all in good fun.
> \*"I created this game to remember that life is random — but it's what you make of it."\*
> — Blake Evan Davis, Creator

🎯 Features
⚔️ Turn-based combat — fight monsters ranging from Gross Goblins to Three Headed Dragons
🗺️ Fast Travel — explore locations like Cancer City, Sticky Swamp, and Ancient Castle
🏥 Hospital system — get patched up, fully healed, or just insulted by Dr. ???
📋 Job Board — take jobs like Town Crier ("Stand in the square and yell things. Nobody will listen. That's the job.")
🛒 Shop & Trading — buy weapons, armor, and potions
🏟️ Arena mode — Easy to Legendary difficulty
⭐ Perks system — customize your build
📜 Quests — structured objectives beyond the main game
🔨 Crafting & Smithing — build and upgrade your gear
💾 Save system — your progress persists
📈 Level & XP system — grow stronger as you play
😂 Unhinged humor — written with real friends, for real laughs
🤖 AI Dialog System (coming soon) — type anything, shape your own story
